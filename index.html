USB Dongle

Few routers can capable to get internet connectivity from the 3G USB dongle and as well as ethernet / ADSL cable.
With these types of routers if cable internet connection lost,
you can get internet connectivity from 3G USB dongle by connecting to the router.

Set-up process

  • Connect the 3G dongle in USB slot.
  • Connect the router with LAN wire that you get with the router. Router cable output wire goes as input wire to Laptop.
  • Open in the browser the link http://192.168.1.1/ similar to it. Check your router installation guide on more details.
  • You can see router configuration page and specify your ISP provider
  • Enter required fields and set a password.

How to connect USB Dongle

3G routers allow to connect USB dongle to the router just like how we connect cable connection to router.
User can into connect USB dongle as internet connection to the router and use the internet over Wi-Fi on multiple devices.

Sharing a network connection steps in windows

  • Step 1 - Open the Windows Settings. (Click the Windows Menu, then click the Settings icon.)
  • Step 2 - Select the Network & Internet option.
  • Step 3 - Select Change adapter options.
  • Step 4 - In the Network Connections window, find the network you wish to share,
    right-click the network, and select Properties (this would generally be a Wi-Fi connection).
  • Step 5 - Select the Sharing tab.
  • Step 6 - Click the checkbox for "Allow other network users to connect through this computer’s Internet connection.”
  • Step 7 - In the “Home networking connection:” drop-down, select the Ethernet adapter to which you are connecting your ALTA Ethernet Gateway.
  • Step 8 - Click the Ok button.
  • Step 9 - Connect the Ethernet port of your gateway to the Ethernet port on the PC using a standard Ethernet cable.
  • Step 10 - Power the ALTA Ethernet Gateway up.

Network types

1.LAN is a group of network devices that allow communication between connected devices. The private ownership has control over the local area network rather than the public. LAN has a shorter propagation delay than MAN as well as WAN.
It covers the smallest area such as College, School Hospital and so on.

2.WAN covers a larger area than LAN as well as MAN such as Country/Continent etc. WAN is expensive and should or might not be owned by one organization.
PSTN or Satellite medium is used for wide area networks.
